\subsection{Detailed Description}
This is a pared-\/down version of the main class, \hyperlink{classgov_1_1fnal_1_1ppd_1_1dd_1_1ChannelSelector}{Channel\-Selector}, that only allows you to send a very restricted subset of channels to the \char`\"{}alternate frames\char`\"{} at each display. The Use Case is that the channel will be a text web page of lab-\/wide announcements. A separate mechanism will be provided to change the text in this web page.
