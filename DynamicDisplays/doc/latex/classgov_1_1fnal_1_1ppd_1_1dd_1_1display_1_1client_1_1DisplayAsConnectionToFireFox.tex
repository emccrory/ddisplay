\subsubsection*{Transition to Selenium}

Over the years, the functionality of the system has been improved and extended quite a lot. Not (June 2018) that we are transitioning away from the \char`\"{}pmorch\char`\"{} plugin to Selenium, a lot of that built-\/up functionality and reliability needs to be moved to either this base class, or to the base class of the attribute browser\-Instance. 

So, only the functionality specific to the Firefox browser and the pmorch plugin should remain here.
