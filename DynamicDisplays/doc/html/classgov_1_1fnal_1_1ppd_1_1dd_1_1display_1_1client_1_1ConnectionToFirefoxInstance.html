Detailed Description

Connects to an instance of Firefox browser and sends JavaScript instructions to it. This is the "lowest" level class in the suite.

(November, 2015) – Need to be careful about not letting different threads stomp on the activities of this class. Up until now, there were "synchronize" blocks on the instance(s) of this class.

Elliott McCrory, Fermilab AD/Instrumentation
