gov.fnal.ppd.dd.display.client.simplified.SimplifiedBrowserConnection Class Reference

Detailed Description

A connection to the browser through the operating system. That is, when the channel is changed, we have to restart the browser at the new locations. It is thought that this sort of browser connection would be used only when (a) the channel pretty much never changes, and/or (b) the target web page is an http (non-secure) page. The channel change does NOT happen here.
