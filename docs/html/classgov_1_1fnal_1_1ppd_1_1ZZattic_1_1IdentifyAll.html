gov.fnal.ppd.ZZattic.IdentifyAll Class Reference

Detailed Description

Let the user ask each Display in the Dynamic Display system to go to the "Identify" screen. Since this Channel is designed to only be on the screen for a fraction of a minute, the Display will go back to its "regularly scheduled program" after a while.
