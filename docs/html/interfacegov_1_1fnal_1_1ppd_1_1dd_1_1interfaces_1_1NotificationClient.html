gov.fnal.ppd.dd.interfaces.NotificationClient Interface Reference

Detailed Description

This interface allows the client to show update information during the process of doing the update. There are two concrete classes: A GUI class and a text class. The GUI class is used by the ChannelSelector and the Displays, and the text client is used by the MessagingServer.
