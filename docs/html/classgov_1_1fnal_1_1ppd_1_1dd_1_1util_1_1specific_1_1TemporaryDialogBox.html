gov.fnal.ppd.dd.util.specific.TemporaryDialogBox Class Reference

Detailed Description

Creates a borderless pop-up window. Used by gov.fnal.ppd.dd.ChannelSelector, and others, to let the user know that something is happening. Disappears after a short interval.
