gov.fnal.ppd.ZZattic.EmptyChannel Class Reference

Detailed Description

An empty channel for use by these XML classes - This used to be the name of the class, but it was changed to XmlDerivedChannel. This class has been reintroduced so that channels stored in the table DefaultChannels will still work.
