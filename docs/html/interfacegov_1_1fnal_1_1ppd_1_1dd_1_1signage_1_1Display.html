gov.fnal.ppd.dd.signage.Display Interface Reference

Detailed Description

The signature of an electronic content displayer in the Fermilab Dynamic Displays system. This is implemented as a physical monitor attached to a small Linux-based PC, or as an abstraction for that physical configuration within the GUI, a.k.a., ChannelSelector.

Author
Elliott McCrory, Fermilab AD/Instrumentation
