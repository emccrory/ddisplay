\subsection{Detailed Description}
The bare-\/bones implementation of a server that will distribute messages to clients that are subscribed to topics.

This version of the server can only exist within the J\-V\-M where everything is running. The implementation in the Dynamic Displays system would need to have this class be in a separate J\-V\-M, on a different node on the Internet.

The way I would probably implement this is the same way that Displays were implemented -\/ with a Facade within the J\-V\-M that the client connects to, and them that Facade will have the logic to contact the server on the Internet.
