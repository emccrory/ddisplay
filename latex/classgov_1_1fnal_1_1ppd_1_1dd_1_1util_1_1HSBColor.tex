\subsection{Detailed Description}
Generate a set of colors that are (in theory) as distinct as possible from each other.

This is a work in progress. When the number of colors get only slightly large (8), it is quite possible to sort the colors so that there are very similar colors right next to each other. A final scheme for distinguishing colors will probably have to include some words that one can associate with the color for unambiguous differences.
