Detailed Description

On the computer with the channel selector, this is the way you talk to a remote Dynamic Display. This classed is used in the GUI as the Facade between the GUI and the actual display. That actual display sits on the other side of the network connection, through the messaging server.

Author
Elliott McCrory, Fermilab AD/Instrumentation, First version: 2013
