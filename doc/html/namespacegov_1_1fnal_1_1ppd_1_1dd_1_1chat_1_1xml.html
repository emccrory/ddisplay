Detailed Description

Fermilab's Dynamic Displays System – Implementation of XML documents for the chatting

At this time (September, 2016), this is an incomplete (test) implementation of the classes to do the messaging in the system. The idea is that these classes would only output XML documents, and these XML documents are the actual information that is transmitted over the Internet between the clients. Currently, Java Objects are streamed between the clients.

It is thought that this implementation will be necessary if/when we have an Android implementation of the channel selector
