\begin{DoxyItemize}
\item (client) -- the machine that tells the displays what to show asynchronous message from a human, or from a channel script. This client receives a \char`\"{}ready\char`\"{} message, and the it can send a U\-R\-L to the server 
\item (server) -- the machine that controls a display. This server waits for a U\-R\-L from the client. When it gets it and displays the U\-R\-L successfully, it replies, \char`\"{}\-Ready\char`\"{}. 
\end{DoxyItemize}
