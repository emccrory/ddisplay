\subsection{Detailed Description}
This is the protocol for the messages between a Channel Changer and a Display. 

{\itshape D\-C} stands for {\itshape Display Communications} 

The new ideas here (and what differentiates it from the class of the same name in the parent package) are\-: 
\begin{DoxyEnumerate}
\item All of the protocol is handled here, including deciphering of the signature. 
\item It is compatible with A\-S\-C\-I\-I-\/only communication over the wire (via an X\-M\-L document) 
\end{DoxyEnumerate}
